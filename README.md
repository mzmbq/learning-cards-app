# learning-cards-app

## Running the Application

### Docker

1. **Clone the repository**

   ```bash
   git clone https://github.com/mzmbq/learning-cards-app.git
   cd learning-cards-app
   ```

2. **Run Docker Compose**

   ```bash
   docker compose up
   ```

3. **Access at [http://localhost:8080](http://localhost:8080)**
